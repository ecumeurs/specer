### Feature: Analytics Engine

#### Context, Aim & Integration

The analytics engine collects and processes user behavior data for real-time and historical insights.

#### Technical Requirements

- Event tracking via JavaScript SDK
- Data warehouse: ClickHouse
- Batch processing: hourly aggregations
- Real-time streaming via Apache Kafka (new)
- Dashboard UI using Grafana (new)

#### Constraints

- Data retention: 90 days
- PII anonymization required
- Real-time latency: < 5 seconds (new)
