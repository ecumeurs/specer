### Feature: Database Layer

#### Context, Aim & Integration

The database layer provides persistent storage for all application entities.

#### Technical Requirements

- PostgreSQL 15+ required
- Connection pooling via PgBouncer
- Automatic migrations on startup

#### Data Layer

| Entity | Table | Primary Key |
|--------|-------|-------------|
| User | users | uuid |
| Session | sessions | uuid |
| Document | documents | uuid |

#### Constraints

- All timestamps in UTC
- Soft deletes for audit trail
- Maximum 100 connections per pool
